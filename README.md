# webpage-shwoing-Currencyexchange-using-api
This is a webpage that shows and calculates the currency exchanges, it is mainly coded in HTML, CSS, DJANGO, JQUERY, AJAX while using API available on the internet. NOTE: The templates used for the UI exists on the internet but are modified according to the needs.
NOTE
run the collectstatic command after pulling the git so that it startsup properly.

The following webpage has been developed in the linux environment using Django as the server side language.
for now the currencys are limited as the api that provides all the information is paid and hence all the data that is available freely is used.

NOTE
you should have django installed in your computer along with an internet connection for the correct functioning of the project.
